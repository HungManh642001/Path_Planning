# 🎨 Interactive GUI - Ready to Use! 🚀

## ⚡ Get Started in 30 Seconds

### Launch the GUI
```bash
python launch_gui.py
```

### That's it! 
A window opens with:
- **Left panel:** Controls for creating scenarios
- **Right panel:** Map where you draw obstacles
- **Status panel:** Real-time feedback

---

## 🎯 3 Simple Steps

### Step 1: Set Points
1. Click **"🎯 Select Launch Point"** button
2. Click anywhere on the map
3. Do the same for **"🎯 Select Target Point"**

### Step 2: Draw Obstacles (Optional)
1. Click **"🏝️ Draw Island"** → click vertices → close polygon
2. Click **"🛡️ Draw SAM Site"** → click center → move mouse → click again

### Step 3: Run Planning
1. Click **"▶️ Run Path Planning"**
2. See results in new window!
