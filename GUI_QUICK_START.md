# 🎨 Interactive Scenario Builder - Quick Start

## What is it?
An interactive **point-and-click GUI** to create and test custom path planning scenarios.

## How to Run
```bash
python gui_scenario_builder.py
```

## 3 Simple Steps

### Step 1: Set Points
- Click **"🎯 Select Launch Point"** → click on map
- Click **"🎯 Select Target Point"** → click on map

### Step 2: Draw Obstacles  
- Click **"🏝️ Draw Island"** → click vertices → click near first vertex to close
- Click **"🛡️ Draw SAM Site"** → click center → move mouse → click to confirm

### Step 3: Run Planning
- Click **"▶️ Run Path Planning"** 
- See results instantly in new window

## Map Legend
- 🟢 **Green Triangle** = Start
- 🔴 **Red Star** = Target
- 🟤 **Brown Polygon** = Island
- 🔴 **Red Circle** = SAM Defense

## Features
✅ Point-and-click interface  
✅ Real-time obstacle drawing  
✅ Instant path planning execution  
✅ Live status updates
